# Client Lead Automation Pipeline

A Flutter application designed for managing and automating client leads, inspired by n8n workflows. It features a dashboard for key metrics, a Kanban-style pipeline for moving leads through different stages, and a detailed list view for managing individual lead data.

## Features

*   **Dashboard**: Overview of total leads, pipeline value, and recent activity.
*   **Pipeline View**: Interactive drag-and-drop or status-based pipeline to track leads across stages (New, Contacted, Qualified, Proposal, Won).
*   **Leads Management**: A comprehensive list view of all leads with search and filtering capabilities.
*   **Responsive Design**: Works seamlessly across mobile, tablet, and desktop environments.

## Tech Stack

*   Framework: Flutter
*   Design System: Material 3
*   Architecture: Standard Flutter UI patterns with models and mock data.

## Setup and Run

1.  Ensure you have the Flutter SDK installed and configured.
2.  Run `flutter pub get` to fetch dependencies.
3.  Run `flutter run` to launch the application on your preferred device or emulator.
